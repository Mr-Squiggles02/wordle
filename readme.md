# Hello, this code is based on the game Wordle
## To be able to use this code, you must follow these steps
### 1.  First you must create a virtual environment for this code. To do this you must:
- Open up a new Terminal
- Type in the command `python -m venv <"chosen name" e.g. venv>`
- After that, you must acitvate the virtual envirment so that you are able to make changes without affecting other projects.
### 2. Then you must activate the virtual environment you just created:
- On a new line in the terminal type in the command `source venv/bin/activate`
- After you type this you should see the name of your virtual environment in brackets at the start of a new line
### 3. Installing the libraries:
- After this you need to install all the libaries, you do this by installing the requirments.txt using another command on the new line of the terminal
- Type in the command `pip install -r requirements.txt`
## After following these steps, you should now be able to run my wordle code :)
